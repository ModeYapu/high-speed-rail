# 更新日志

## [v1.1.0] - 2026-03-03

### 🎵 新增：音效系统

#### 核心功能
- **AudioManager 类**：完整的音频管理系统
  - 支持主音量、音效音量、音乐音量独立控制
  - 静音/取消静音功能
  - 音效暂停/恢复

#### 音效类型
1. **引擎声** (`engine`)
   - 基于速度的动态频率调整（60-240Hz）
   - 双振荡器叠加（sawtooth + square）
   - 音量随速度变化
   - 实时更新，平滑过渡

2. **刹车声** (`brake`)
   - 金属摩擦效果
   - 频率衰减包络（200Hz → 50Hz）
   - 低通滤波器增强真实感
   - 1.5秒持续时间

3. **鸣笛声** (`horn`)
   - 双音叠加（大三度和弦）
   - 2秒持续时间
   - 音量包络（淡入淡出）

4. **风声** (`wind`)
   - 白噪声 + 带通滤波器
   - 频率和音量随速度动态调整
   - 50km/h以上自动启动

5. **雨声** (`rain`)
   - 粉红噪声模拟雨滴
   - 强度可调（0-1）
   - 恶劣天气模式自动启动

6. **隧道回音** (`tunnel`)
   - 延迟效果器
   - 反馈回路
   - 低通滤波

7. **车站广播** (`station`)
   - 三音提示序列
   - 关卡完成时播放

8. **警报声** (`alert`)
   - 交替频率（800/600Hz）
   - 超速警告时触发

9. **点击声** (`click`)
   - 短促的UI反馈音
   - 50ms持续时间

#### 游戏集成
- **Game 类**：
  - 新增 `audio` 属性
  - 新增 `createAudio()` 方法
  - 新增 `updateAudio()` 方法
  - 新增 `toggleAudio()` 方法
  - 在 `animate()` 循环中自动更新音效
  - 暂停时停止所有音效
  - 关卡完成时播放车站广播

- **Controls 类**：
  - 紧急制动时播放刹车声
  - 普通制动时播放刹车声（速度>20km/h）
  - 新增音效控制按钮事件

#### UI 更新
- 新增音效控制按钮（🔊 音效 / 🔇 静音）
- 按钮状态实时反馈

#### 技术细节
- **实现方式**：Web Audio API
- **音频类型**：纯合成音效（无外部文件）
- **性能优化**：
  - 引擎声和风声只在需要时启动
  - 停止时立即释放资源
  - 音量平滑过渡避免爆音

#### 文件变更
- 新增：`js/audio.js`（450行）
- 修改：`js/game.js`
  - 集成音效系统
  - 添加音效更新逻辑
- 修改：`js/controls.js`
  - 添加刹车音效触发
  - 添加音效按钮事件
- 修改：`index.html`
  - 引入 audio.js
  - 添加音效控制按钮
