# 🚄 高铁模拟驾驶游戏

中国高速铁路模拟驾驶游戏 - 和谐号 & 复兴号

## 功能特点

- ✅ 真实物理引擎模拟
- ✅ 和谐号 CRH380A & 复兴号 CR400AF
- ✅ 多种游戏模式
- ✅ 动态天气系统
- ✅ 精美的3D画面
- ✅ 完整的HUD界面

## 游戏模式

1. **自由驾驶** - 无限探索，享受驾驶乐趣
2. **准点到达** - 按时到达目的地
3. **恶劣天气** - 雨雪天气安全驾驶
4. **紧急任务** - 应对突发情况

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 加速 |
| S / ↓ | 减速/制动 |
| A / ← | 左转 |
| D / → | 右转 |
| 空格 | 紧急制动 |
| ESC | 暂停游戏 |

## 技术栈

- Three.js - 3D渲染引擎
- WebGPU兼容 - 高性能渲染
- 纯前端实现 - 无需后端

## 运行方式

### 方式1: 直接打开
用浏览器打开 `index.html` 即可

### 方式2: 本地服务器
```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve
```

然后访问 http://localhost:8080

## 列车数据

### 和谐号 CRH380A
- 最高时速: 380 km/h
- 加速度: 0.8 m/s²
- 特点: 平稳舒适

### 复兴号 CR400AF
- 最高时速: 400 km/h
- 加速度: 0.9 m/s²
- 特点: 中国标准，世界领先
